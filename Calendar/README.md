# Calendar

自分用のミニマルなカレンダーアプリ（Electron製、Windowsスタンドアロン）。

## 起動方法

```powershell
cd "c:\Users\yasuh\AIの作業場\Calendar"
npm start
```

数秒で「Calendar」という名前のウィンドウが開きます。

## 主な操作

- マウスホイールで上下スクロール → 月ヘッダーが自動で切り替わる
- 月ヘッダー中央（例: `2026.7`）をクリック → 表示位置を正規化。`<` `>` で前後の月へ
- 空いている日付をクリック → 予定名を入力してEnter → パレットから色を選択
- 予定が入っている日を約0.5秒長押し → 詳細画面（時間・メモ・色変更・削除）
- 何もない日を押したままドラッグ → 複数日にまたがる予定（帯）を作成
- 詳細画面の色スワッチを長押し → OS標準のカラーピッカーでその色を再設定（土日祝グレーの枠だけ固定）

## データ保存先

`%APPDATA%\Calendar\calendar-data.json`（自動保存、ローカルのみ・クラウド同期なし）
