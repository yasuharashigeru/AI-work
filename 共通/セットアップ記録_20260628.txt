【目的】
テキストファイルの内容を読み込み、図解入りPowerPoint（.pptx）を
自動生成するための環境構築。

----------------------------------------------------------------
■ インストールしたアプリ・ライブラリ
----------------------------------------------------------------

① Python 3.12.10
  - インストール方法：winget install Python.Python.3.12
  - インストール場所：C:\Users\yasuh\AppData\Local\Programs\Python\Python312\
  - 用途：pptx自動生成スクリプトの実行環境

② python-pptx（Pythonライブラリ）
  - インストール方法：python -m pip install python-pptx
  - インストール場所：C:\Users\yasuh\AppData\Local\Programs\Python\Python312\Lib\site-packages\pptx\
  - 用途：PythonからPowerPoint（.pptx）ファイルを生成するライブラリ

③ LibreOffice 26.2.4.2
  - インストール方法：winget install TheDocumentFoundation.LibreOffice
  - インストール場所：C:\Program Files\LibreOffice\
  - 用途：.pptxファイルを開いて閲覧・編集するオフィスソフト
  - 起動方法：.pptxファイルをダブルクリックで開ける
