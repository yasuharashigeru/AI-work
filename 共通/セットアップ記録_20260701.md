## 2026年7月1日：Codexコードレビュー環境

**目的：** Claude Code（VSCode拡張）からOpenAI Codexを呼び出してコードレビューを行えるようにするための環境構築。

### インストールしたアプリ・ライブラリ

| # | 名前 | バージョン | インストール方法 |
|---|------|-----------|----------------|
| ① | Node.js（LTS） | 24.18.0 | `winget install OpenJS.NodeJS.LTS` |
| ② | Git | 2.54.0 | `winget install Git.Git` |
| ③ | Claude Code CLI | 2.1.196 | `npm install -g @anthropic-ai/claude-code` |
| ④ | codex-plugin-cc | 1.0.5 | Claude Code CLI内の `/plugin` コマンドで導入 |
| ⑤ | @openai/codex | 最新 | `/codex:setup` 実行時に自動インストール |

> ⚠️ **Claude Code CLI 初回起動時の注意：** PowerShellの実行ポリシー変更が必要
> ```
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```

**codex-plugin-cc のインストール手順（Claude Code CLI内で順番に実行）：**
```
/plugin marketplace add openai/codex-plugin-cc
/plugin install codex@openai-codex    ← user scope を選択
/reload-plugins
/codex:setup                          ← Codex CLIのインストールも実行
!codex login                          ← OpenAIアカウントと紐付け
```

> ⚠️ `/plugin` コマンドは **Claude Code CLIでのみ動作**（VSCode拡張では使えない）

### セットアップした環境

**⑥ AIの作業場フォルダをgitリポジトリ化**
- 場所：`C:\Users\yasuh\OneDrive\デスクトップ\AIの作業場\`
- 方法：`git init` → `git add .` → `git commit -m "initial"`
- Codexのレビューはgitの差分（diff）を対象とするため必須

### 次回以降の使い方

1. VSCodeのターミナルで `claude` と入力してClaude Code CLIを起動
2. レビューしたいファイルを編集・保存
3. 以下のコマンドでレビュー実行

| コマンド | 内容 |
|---------|------|
| `/codex:review` | 変更箇所の標準レビュー |
| `/codex:review 収支アプリ/app_standard.html` | 特定ファイルのみレビュー |
| `/codex:adversarial-review` | 設計の問題点を厳しく指摘 |
| `/codex:rescue` | バグ調査・修正を丸投げ |

> ⚠️ Codexのレビューはgitの**差分が対象**。変更なしだとレビュー対象ゼロになる
